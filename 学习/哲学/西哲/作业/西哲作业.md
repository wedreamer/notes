#### 名词解释
##### 泰勒斯

```txt
    泰勒斯通常被认为是西方哲学史上的第一位哲学家. 他提出了万物有灵和水本原论.
    泰勒斯从对宇宙, 自然万物的观察以及思考, 大地浮在水上, 水形成了风, 云, 雷, 火等世间万物, 提出了万物都具有灵魂, 具有灵动的特性, 这里的万物有灵中的灵是指湿气. 同时进一步提出了水本原论, 即认为宇宙的本原是水, 水作为宇宙的基质, 构成了宇宙万物. 
    泰勒斯从自身生活环境, 自然世界进行观察以及思考, 同时可能也受到了古希腊神化的影响, 因为古希腊神话中就是由水神进行世界创造, 从而提出了宇宙究竟由何构成, 也就是宇宙本原的问题, 并进行了回答即水本原论. 这个过程体现了哲学的基本研究方法, 即提出问题并进行作答的. 同时泰勒斯脱离古希腊神话中神创造世界的观念, 从自己的思考观察给出了水本原的答案, 这也标志着思想脱离了旧有神话观念的束缚, 实现了从神性到理性的转变.
```

##### 水是万物的本原

```txt
    泰勒斯作为西方哲学史上的第一位哲学家提出了万物有灵和水本原论的哲学观点.
    他从自身的生活环境即自然世界观察到大地浮在水上, 水经过各种转化形成了风, 雷, 雨, 电等世间万物, 认为万物都具有灵魂, 都具有灵动的特性即万物有灵, 这里的灵魂是指湿气. 进而认为宇宙组成的基本物质就是水, 水经过各种转化构成了宇宙万物, 从而提出了水本原论.
    泰勒斯从自然环境出发, 依靠自己的观察总结和思考回答了宇宙的构成基质, 即宇宙的本原是什么? 提出问题以及回答问题的这种方式, 体现出了基本的哲学研究方法. 水本原可能跟古希腊的神话有关, 在神话中水神是作为创世的存在. 这个过程还体现出了人类思想开始脱离了旧有神话观念的影响, 依靠自身的理性思考去解决问题, 实现了从神性到理性的转变.
```

##### 无定（阿派朗）

```txt
    继泰勒斯提出水本原之后, 泰勒斯的学生阿那克西曼德提出了无定本原论. 
    无定即无限定, 无定形. 阿那克西曼德认为世界的本原是无定. 阿那克西曼德对于无定的证明认为, 如果世界的本原是有定的, 则有定是生不出世间万物的, 因为万物是无限定的. 因为世界的本原不可能是有定的, 所以世界的本原是无定. 同时认为世界来自无定, 最终复归于无定. 针对无定提出了补偿原则, 即分离生成观, 万物从无定中产生被视为生成, 万物归于无定被视为消亡. 
    无定本原论的提出可以解决阿那克西曼德老师即泰勒斯水本原无法很好解释火相关的世间万物的问题. 但是无定很大程度上只是说明了本原的特性就是无定, 而并非无定就是本原的本身.
```

##### 本原/始基

```txt
    本原是由阿那克西曼德第一次提出的. 
    本原的意义一般有两种, 一种是万物构成的基质, 比如在水本原论中认为世界万物是由水构成的, 火本原论认为世界是由火构成的, 无定本原论认为世界是由无定构成的, 万物生成于无定, 复归于无定, 这里都是做构成万物即宇宙的基质来讲的. 另外一种则是万物运行的法则和规律. 比如赫拉克利特的逻各斯, 通常就是世界的法则, 比如上帝, 比如理念.
```

##### 气本原

```txt
    继米力度学派泰勒斯提出水本原, 阿那克西曼德提出无定本原论之后, 阿那克西曼德的学生提出了气本原论.
    气本原认为气作为宇宙的本原, 气中具有两种特性, 即冷和热. 世间万物都由气构成, 气形成了大地, 风, 火, 雷电等万物. 同时提出了关于气的转化生成观, 气稀疏则为火, 凝聚则为水, 大地, 风, 雷等.
    气本原论继承了泰勒斯的水本原论以及阿那克西曼德的无定本原论, 并进行了发展. 同时气本原论涵盖了泰勒斯水本原以及万物有灵的两个观点, 解决了水本原论对火之类物质直观上很难形成构造关系的问题, 使用气本原则可以进行较好的解释.
```

### 毕达哥拉斯, 赫拉克里克

##### 毕达哥拉斯

```txt
    毕达哥拉斯作为古希腊毕达哥拉斯派的创始人, 核心思想围绕着数和和谐展开, 数对应着智慧, 和谐对应着美好, 他认为数是宇宙的本原, 认为一是完满的, 由一开始而产生万物从而构成宇宙, 这个演化过程就是宇宙论. 由于毕达哥拉斯学派具有宗教信仰的特色, 主要思想中又包含灵魂论. 
    早年毕达哥拉斯擅长数学研究, 同时也接触了米力度学派的各种学说, 他认为在众多本原中数是最能表达出万物的本质的, 万物都能用数来进行表示, 数的属性是公正, 灵魂, 机智. 整个宇宙都是在对数的摹仿. 数量关系就是事物的属性. 数作为万物的本原, 不仅是构成事物的质料, 而且也是事物的属性, 状况和模型. 
    他认为数有两种, 即奇数和偶数, 其中奇数是完满的, 偶数则是不完满的. 自然万物的本原是一元, 从一元产生二元, 从完满的一产生不完满的二, 从而产生各种数目. 从数目产生点, 由点产生线, 由线产生面, 由面产生体, 由体产生出水火土气四大元素, 从而产生出一切物体. 同时他认为天界不过是数和和谐而已.
    毕达哥拉斯认为一切美好的东西都是和谐, 美是和谐, 音乐是和谐, 友爱是和谐. 还有灵魂, 天界等. 同时他也认识到了和谐的对立面, 提出了十对对立范畴.
    毕达哥拉斯有深受东方灵魂不死观念的影响, 认为灵魂是不死, 轮回的, 同一个灵魂可以流动于不同的身体. 而使灵魂净化或者和谐的手段就是通过音乐和哲学.
    虽然毕达哥拉斯提出了数本原论, 同时以数本原为中心形成了宇宙论. 但是混淆了数抽象的概念与有体积的物体, 同时数不能表达感性相关的范畴概念. 由于毕达哥拉斯学派混合政治活动, 宗教信仰, 学术研究于一身, 产生出了具有神秘学倾向的灵魂观.
```

##### 数(毕达哥拉斯)

```txt
    毕达哥拉斯的核心观念就是数和和谐, 其中数对应着智慧, 和谐对应着美好, 由此他提出了数本原, 由数本原而产生了以数为核心的宇宙论.
    毕达哥拉斯作为毕达哥拉斯学派的创始人, 擅长数学研究, 同时也接触了米力度学派的各种学说. 他认为数相对于其他本原如水, 火, 气等可以更好的表达世间万物. 数是先于事物而存在的, 数的属性是公正, 灵魂, 机智. 万物都可以由数来表示, 整个宇宙都是对数的摹仿. 同时数量关系就是事物的属性. 数作为万物的本原, 不仅构成了事物的质料, 而且也是事物的属性, 状况和模型.
    既然数作为宇宙的本原, 宇宙的生成过程又是怎样的？ 毕达哥拉斯认为数有两种, 即奇数和偶数. 其中奇数是完满的, 偶数是缺陷的或者不完满的. 自然万物的本原是一元, 从一元产生二元, 从二元而产生各种数目, 数之后生成点, 点生成线, 线生成面, 面生成体, 体生成水火土气四大元素, 四大元素构成世界万物. 同时他认为天界不过是数和和谐而已.
    毕达哥拉斯认为一切美好的东西都是和谐, 美是和谐, 友爱是和谐, 音乐是和谐, 天界, 灵魂都是和谐的. 同时他也意识到了和谐的对立面, 从而提出了十对对立范畴.
    数本原论对比米力度学派的物质本原论显然更具有抽象意义, 但是从数生出点, 线, 面到生成四大元素的过程中, 毕达哥拉斯混淆了抽象和实在的物质, 同时数本原并不能表达感性范畴中的事物. 
```

##### 赫拉克里克

```txt
    继承米力度学派的有形本原论, 赫拉克里克提出了火本原论. 以及对物质本原的补偿说明, 从而提出了逻各斯, 辩证法. 辩证法则继承自毕达哥拉斯学派.
    火本原论认为世界的本原是由火构成的. 这个万物自同的宇宙, 既不是任何神, 也不是任何人所创造的, 它过去是, 现在是, 将来也是是一团永恒的活生生的火, 在一定的分寸上燃烧, 在一定分寸的上熄灭. 首先赫拉克利特否认神创世, 以及人创世. 认为宇宙的本原是活生生火, 其中活生生的火中的活生生则表示变动不局, 无时无刻不在变化, 唯一不变的就是变化本身, 意味着万物皆流, 无物常住. 其中继承了无定论中无定的特性. 而一定的分寸则在表示变化的规律, 尺度或者法则, 即逻各斯, 万物按照这样的法则进行流变, 也就是燃烧和熄灭. 针对本原的转化补偿, 赫拉克利特认为火化生万物, 万物复归于火. 
    逻各斯本意为言语, 还指规律和法则, 在火本原中即为一定的分寸. 即为世界万物在燃烧和熄灭时的法则和规律, 也就是万物流变的规律和法则.
    赫拉克利特认识到了对立面之间的斗争和统一是运动和变化的根源，由火本原论强调火的生生不息，肯定了一切皆流，无物流住．认为人不能两次踏入同一条河流，因为河流一直在变化，无时无刻不在变化．他意识到了对立之间的关系，认为相关的力量构成和谐, 就像弓和琴一样; 生死, 老少, 睡醒是一样的, 向上的路和向下的路是同一条;　对于斗争和统一，他更关注斗争，认为战争是普遍的, 正义就是斗争, 万物按照斗争和必然性而生成; 可能是由于希腊民族古老信念的影响，赫拉克利特认为斗争的结果即和谐和统一是万物变化的归宿，即把变化的总趋势理解为循环和平衡．
    赫拉克里克继承了米力都学派的本原特点, 辩证法则继承于毕达哥拉斯学派. 在提出本原学说的同时提出了逻各斯, 使得知识成为可能. 逻各斯在后来的巴门尼德、阿那克萨戈拉、柏拉图、亚里士多德等人的哲学中，都有或明或暗的反映。更重要的是，在后来西方哲学、神学和科学的发展过程中，逻各斯概念及其衍生的逻辑（logic）一直起着重要的作用.
```

##### 逻各斯

```txt
    赫拉克利特提出了火本原论，火本原认为一切皆流，无物常住．针对变化的尺度而提出了逻各斯．
    对于火本原论，赫拉克利特认为，这个万物自同的宇宙，既不是由任何神，也不是由任何人所创造的，它过去是，现在是，将来也是一团永恒的活生生的火，在一定的分寸上燃烧，在一定的分寸上熄灭．首先赫拉克利特否认了神创世说，以及人创世说．认为宇宙的本原就是火，而且是活生生的火，活生生则意味着万物永恒在变化中，唯有变化本身是不变的，即一切皆流，无物常住．同时这个变化受到了一定分寸的约束，也就是变化的规律，规则，即逻各斯．针对火本原，赫拉克利特还认为万物由火生出，最终还复归于火．即是火本原的补偿原则．
    逻各斯本意为语言，同时也有规则，法则，尺度的意思．同时也是万物变化所遵循的尺度，即一定分寸．
    赫拉特利特提出的逻各斯使知识成为可能, 逻各斯在后来的巴门尼德，阿那克萨戈拉，柏拉图，亚里士多德等人的哲学中，都有或明或暗的反映．更为重要的是，在后来的西方哲学，神学和科学的发展中，逻各斯及其衍生出的逻辑一直起着重要的作用．
```
